How Link.run Works?

  • Create Simple Names for
    Long URLs

    Naming URLs makes it easy to remember
    them. For example, http://link.run/agenda

  • Access Named Links Created
    by Colleagues

    When your colleagues create simple names
    for long URLs, you can also access those links
    using simple names. For example: HR creates
    a named link for 401k at http://link.run/401k

  • Names are Private to Your
    Workplace

    Everyone in your company’s @xyz.com
    domain can access these links
